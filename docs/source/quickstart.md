# Quick Start Guide

This guide will get you up and running with DnD Agent in just a few minutes.
## Prerequisites

- Python 3.10 or higher


## Installation

### Install from PyPI

```bash
pip install dnd_agent
```

### Install from Source

```bash
git clone https://github.com/redam94/dnd_agent
cd dnd_agent
pip install -e ".[dev]"
```
